# Global Page Analysis

### Same appearance
- Verify overall page layout structure matches
- Verify major visual blocks appear in corresponding positions
- Ignore numeric/date/value differences that can vary over time

### Content completeness
- Verify no major section is missing in the test page
- Verify no major section is unexpectedly added only in the test page
- Verify headers and labels are present and readable
- Language is chinese

### Visual quality
- Verify no rendering artifacts, clipping, or truncation are visible
- Verify spacing and alignment are not clearly broken
- Verify typography style looks consistent
